# VoicePost

AI-powered LinkedIn content engine for enterprise B2B.

## Features

- **Post Generation**: Create scroll-stopping LinkedIn posts using AI
- **Multiple Frameworks**: Stanley's 8-step, Brandon's 3-bucket, Daniel's 7-step
- **Image Prompts**: Auto-generated image prompts for higher engagement
- **Hook Methodology**: Built-in hook formulas and rules
- **Profile Optimizer**: LinkedIn profile optimization checklist
- **Content Calendar**: Weekly content planning

## Setup

1. Clone the repo
2. Copy `.env.local.example` to `.env.local` and add your MIMO API key
3. Install dependencies: `npm install`
4. Run the dev server: `npm run dev`

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- TypeScript
- MIMO API (Xiaomi AI)

## Deployment

Deploy to Vercel:

```bash
vercel
```

Add your `MIMO_API_KEY` in the Vercel dashboard under Environment Variables.
